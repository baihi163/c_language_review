## 第 11 题

![填空11](images/实验9/07_填空11阶乘static.png)
![填空11-12](images/实验9/08_填空11-12.png)

用 `static int s=1` 连续调用 `func(1)…func(10)` 输出 1!~10!。

| 空 | 正确答案 | 你的答案 |
|----|----------|----------|
| ① printf | **`"%d!=%d\n", i, func(i)`** | 缺第一个 `i` |
| ② 累乘 | **`s = s * n`** 或 `s *= n` | `s=s*i` ✗（`i` 是 main 里的变量，func 里不认识） |

![第11题②③纠错](images/实验9/09_填空11纠错2-3.png)

**【⚠️ 避坑】作用域陷阱**：形参叫 `n`，函数体内只能用 `n`，不能写 `main` 里的 `i`。

### (1) 改成 `static int s; s=1;` 会怎样？

每次调用都把 `s` 重置为 1，只能算 `n×1`，**10! 行变成 `10!=10`**。

| 空 | 正确答案 | 你的答案 |
|----|----------|----------|
| ③ 10! 输出 | **10** | 1 |
| ④ 原因 | **B. 每次调用都重置 static** | B ✓ |

### (2) 保持原定义，直接调用 `func(10)`（无循环）

只执行一次 `s=1*10`，结果是 **10**，不是 3628800。

| 空 | 正确答案 | 你的答案 |
|----|----------|----------|
| ⑤ 结果 | **10** | 10 ✓ |
| ⑥ 原因 | **B. 必须从 1 开始依次调用** | A |

## 第 13 题

![填空13](images/实验9/14_填空13递归阶乘.png)
![填空13纠错2-4](images/实验9/17_填空13纠错2-4.png)
![填空13纠错5](images/实验9/18_填空13纠错5else.png)

用**递归**实现阶乘，循环输出 1!~10!。

| 空 | 正确答案 | 你的答案 |
|----|----------|----------|
| ① printf 参数 | **`func(i)`** | `i,func(i)` ✓（① 在 `"%d!=%d\n",` 后面） |
| ② 递归边界 | **`n == 1) s = 1`** | `(n=1) s=1` ✗ 要用 `==` 比较 |
| ③ 递归公式 | **`s = n * func(n - 1)`** | `s=s*n` ✗ 那是循环写法 |
| ④ 直接 func(10) | **`3628800`** | `10` ✗ 别和 static 那题混了 |
| ⑤ 去掉 else 后果 | **A. 无终止条件，无限递归** | B ✗ |

```c
int func(int n) {
    int s;
    if (n == 1)          // ② 边界：必须写 ==
        s = 1;
    else
        s = n * func(n-1); // ③ 自己调自己
    return s;
}
```

**递归两大护法**：① 边界 `if(n==1)` ② 递推 `else s=n*func(n-1)`。没有 `else`，`n=1` 后还会执行 `func(0)→func(-1)…` → **栈溢出**。

## 第 15 题

![填空15](images/实验9/20_填空15质数判断.png)
![填空15纠错](images/实验9/21_填空15纠错5空.png)

自定义 `is_prime`，质数返回 1，否则返回 0。

| 空 | 正确答案 | 你的答案 |
|----|----------|----------|
| ① main 调用 | **`is_prime(a) == 1`** | ✓ |
| ② 函数首部 | **`int is_prime(int n)`** | ✓ |
| ③ for 循环 | **`i = 2; i <= n/2; i++`** | ✓ |
| ④ 发现因子 | **`if(n % i == 0)`** | ✓ |
| ⑤ 循环后判断 | **`if(i <= n/2)`** | `if(i==n/2)` ✗ |

**⑤ 为何用 `<=` 不是 `==`？** 以 n=9 为例：i=3 时 `9%3==0` 触发 `break`，此时 i=3，而 n/2=4。`3==4` 为假会误判 9 为质数；`3<=4` 为真才正确返回 0。

## 第 16 题

![填空16](images/实验9/22_填空16幂函数.png)
![填空16纠错](images/实验9/23_填空16纠错3-4.png)

自定义 `power(x, y)` 用循环实现 x 的 y 次方。

| 空 | 正确答案 | 你的答案 |
|----|----------|----------|
| ① 读底数 | **`scanf("%f", &a);`** | ✓ |
| ② 调用并输出 | **`power(a, b)`** | ✓ |
| ③ 函数定义 | **`float power(float x, int y)`** | 注意拼写 `float` 不能写成 `floar` |
| ④ for 初始化 | **`i = 0, z = 1`** | 只写 `i=0` ✗ `z` 也要初始化为 1 |
| ⑤ 累乘 | **`z = z * x`** | ✓ |

**【⚠️ 避坑】未初始化局部变量是垃圾值**，`z=z*x` 前必须 `z=1`，否则结果随机。

```c
float power(float x, int y) {
    float z;
    int i;
    for (i = 0, z = 1; i < y; i++)   // ④ 逗号表达式一次初始化两个
        z = z * x;                    // ⑤
    return z;
}
```
