## 第 1 题

![单选1-2](images/结课练习一/01_单选1-2.png)

```c
main()
{
    int num=0;
    while (num<=2)
    {
        num++; printf("%d ", num);
    }
}
```

| 轮次 | num（判断前） | 动作 | 输出 |
|------|---------------|------|------|
| 1 | 0 ≤ 2 ✓ | num→1 | 1 |
| 2 | 1 ≤ 2 ✓ | num→2 | 2 |
| 3 | 2 ≤ 2 ✓ | num→3 | 3 |
| 4 | 3 ≤ 2 ✗ | 结束 | — |

| 你的答案 | 正确答案 |
|----------|----------|
| A ✓ | **A. 1 2 3** |

## 第 2 题

```c
int y=18, i=0, j, a[6];
do {
    a[i] = y % 2;  i++;
    y = y / 2;
```

## 第 3 题

![单选3-5](images/结课练习一/02_单选3-5.png)

```c
int *f(int *x, int *y)
{
    if (*x < *y) return x;
    else return y;
}
// p=&a(7), q=&b(8), r=f(p,q) → r 指向较小者 a
printf("%d,%d,%d\n", *p, *q, *r);  // 7, 8, 7
```

| 你的答案 | 正确答案 |
|----------|----------|
| C ✓ | **C. 7,8,7** |

函数返回的是**指针**（地址），`r` 与 `p` 同指 `a`，`*r=7`。

## 第 4 题

```c
void fun(char *a, char *b)
{
    a = b;      // 只改了形参副本 a 的指向，p1 不变
    (*a)++;     // 通过 a 改的是 c2：'a' → 'b'
}
// c1='A' 未动，c2='b'
printf("%c%c\n", c1, c2);
```

| 你的答案 | 正确答案 |
|----------|----------|
| A aa | **C. Ab**（或选项中含 `Ab` 的项） |

### ⚠️ 避坑指南

- `a = b` 是**指针副本**重新赋值，不影响 `main` 里的 `p1`
- `(*a)++` 改的是 `a` **当前指向**的内存（即 `c2`）
- 想改 `p1` 的指向需 `char **`，想改 `c1` 需传 `&c1`

## 第 5 题

```c
int f(char *s)
{
    char *p = s;
    while (*p != '\0') p++;
    return (p - s);    // 指针相减 = 元素个数
}
printf("%d\n", f("goodbey"));
```

`"goodbey"` 共 **7** 个字符（g-o-o-d-b-e-y）。

| 答案 |
|------|
| **7** |

`p - s`：两指针间相差的元素个数，即字符串长度。
