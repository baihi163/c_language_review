## 第 3 题

![单选3-5](images/结课练习一/02_单选3-5.png)

```c
int *f(int *x, int *y)
{
    if (*x < *y) return x;
    else return y;
}
// p=&a(7), q=&b(8), r=f(p,q) → r 指向较小者 a
printf("%d,%d,%d\n", *p, *q, *r);  // 7, 8, 7
```

| 你的答案 | 正确答案 |
|----------|----------|
| C ✓ | **C. 7,8,7** |

函数返回的是**指针**（地址），`r` 与 `p` 同指 `a`，`*r=7`。

## 第 4 题

```c
void fun(char *a, char *b)
{
    a = b;      // 只改了形参副本 a 的指向，p1 不变
    (*a)++;     // 通过 a 改的是 c2：'a' → 'b'
}
// c1='A' 未动，c2='b'
printf("%c%c\n", c1, c2);
```

| 你的答案 | 正确答案 |
|----------|----------|
| A aa | **C. Ab**（或选项中含 `Ab` 的项） |

### ⚠️ 避坑指南

- `a = b` 是**指针副本**重新赋值，不影响 `main` 里的 `p1`
- `(*a)++` 改的是 `a` **当前指向**的内存（即 `c2`）
- 想改 `p1` 的指向需 `char **`，想改 `c1` 需传 `&c1`

## 第 5 题

```c
int f(char *s)
{
    char *p = s;
    while (*p != '\0') p++;
    return (p - s);    // 指针相减 = 元素个数
}
printf("%d\n", f("goodbey"));
```

`"goodbey"` 共 **7** 个字符（g-o-o-d-b-e-y）。

| 答案 |
|------|
| **7** |

`p - s`：两指针间相差的元素个数，即字符串长度。

## 复习重点总结

### 1. 指针（错最多，必背）

| 概念 | 记住这句 |
|------|----------|
| 传值 vs 传址 | 形参指针 `p1=p2` 只改局部副本，改不了实参（**4**） |
| `*p` 解引用 | 改的是「房间里的人」，`*p1=*p2` 是**复制内容**不是改指针（**17**） |
| 指针数组 | `int *p[3]` 是**3 个 int 指针**，不是 `int[3]`（**57**） |
| 数组与地址 | `s[1]` 是**元素值**，不是地址；`&s[1]` 才是（**65**） |
| 字符串复制 | `while(*p2++)` 会**漏掉 `\0`**，长度/复制题要看终止符（**66**） |
| 定义里的 `*` | `int *p` 中 `*` 是**类型说明符**的一部分（**113**） |
| 作用域遮蔽 | 局部变量与全局同名时，**局部优先**（**114**） |
| `printf` 与指针 | `*(ptr+2)` 用 `%d` 输出的是**值**（如 16），不是地址（**176**） |

**专题口诀**：`p = q` 换房间号；`*p = 10` 改房间里的人。

---

### 2. 宏定义（预处理陷阱）

```c
#define F(x) P*x*x
F(3+5)  →  3*3+5*3+5 = 29   // 不是 3*(3+5)*(3+5)
```

- 宏 = **无脑文本替换**，不算括号、不先算参数。
- 安全写法：`#define F(x) (P*(x)*(x))`（**153**）

---

### 3. 运算符与表达式

| 陷阱 | 例题要点 |
|------|----------|
| 连续比较 | `a>b>c` 实际是 `(a>b)>c`，结果为 0/1，**不能**当数学连比（**12, 166**） |
| 优先级 | `=` 很低，`--`/`++` 很高；`a=3*5` 先算乘法（**30**） |
| 后缀 `++`/`--` | 表达式**先用旧值**，再自增/自减（**44, 168**） |
| 前缀 vs 后缀 | `++A==B--` 要分步：先算 ++A 和 B-- 的**表达式值**，再比（**155**） |
| 逗号表达式 | 整体值 = **最后一个**子表达式的值（**74**） |
| 位运算 | `~a` 按位取反；`b=~a` 后 `~c` 与 `b` 比较（**150**） |
| 短路 | `&&`/`||` 短路，右侧可能**不执行**（**117**） |
| 左值 | 赋值号左边必须是左值，`y*5=…` 非法（**20**） |

---

### 4. 输入输出（scanf / printf）

| 考点 | 要点 |
|------|------|
| 格式串要**完全匹配** | `"%d,%d"` 输入必须有逗号；`"11 12"` 错（**156, 165, 177**） |
| 宽度限制 | `"%2d%f"` 只读 2 位整数，剩余留给下一项（**162**） |
| `%o` / `%x` | 默认**不带** `0` / `0x` 前缀；要前缀用 `%#o`、`%#x`（**157**） |
| 字符串 `%s` | 不要对数组名再取址 `&s`（**53**） |
| `%d` 打印字符 | 输出的是 **ASCII 码**（如 109），不是字母本身（**133**） |

---

### 5. 循环（while / for / break / continue）

| 陷阱 | 说明 |
|------|------|
| `while(k=0)` | 赋值表达式值为 0，循环**一次都不进**（**31**） |
| `while(n--)` 与 `--n` | 搭配方式决定输出序列（**18**） |
| `continue` | 跳过本次剩余语句，**for 的 i++ 仍会执行**；while 里手写 i++ 可能被跳过 → 死循环（**123**） |
| `break` | 只跳出**最近一层**循环；看清 break 时循环变量是多少（**159**） |
| for 执行顺序 | 条件 → 体 → **更新表达式** → 再条件（**103**） |
| 嵌套循环 | 画表追踪外层/内层各变量（**98, 108**） |

---

### 6. switch 与分支

- **不能**对 float/double 做 switch（**55**）
- 无 `break` 会**穿透**到下一个 case；`default` 位置不限，同样会穿透（**151**）
- `if-else` 嵌套要看清**哪条分支**与其他选项不同（**71**）

---

### 7. 字符串与字符

| 考点 | 要点 |
|------|------|
| `strlen` vs `sizeof` | strlen = 有效字符数（不含 `\0`）；sizeof 含 `\0` 和填充（**48, 75, 86**） |
| 字符常量 | 单引号**只能一个字符**，`'cd'` 非法（**126, 152**） |
| 转义序列 | `\123` 等按规则算长度，不是普通字符（**33**） |
| `strcmp` | 按 ASCII 比，`'D' < 'd'`（**96**） |
| 求长度写法 | `while(*y++); return(y-s-1);` 经典 strlen（**158**） |

---

### 8. 数组与二维数组

- 二维数组行优先：`a[i*m+j]` 是线性下标（**110**）
- 每行字节数 = 列数 × `sizeof(元素)`（**78**）
- 数组名不能做 `s++`（**61**）
- `'d'-'a'` 可作下标，值为 3 → `a[3]` 是第 4 个元素（**175**）

---

### 9. 函数与 static

- **static 局部变量**：只初始化一次，**跨调用保留**（**141**）
- 递归：直接/间接都要能想到调用链（**89**）
- `return` 表达式 vs `void` 函数：带返回值的 return 不能写在 void 里

---

### 10. 进制与常量

- 以 `0` 开头是**八进制**：`012` = 10（**174**）
- `0.1e+6`、 `"\a"`、 `'\011'` 均合法；`'cd'` 不合法（**152**）
