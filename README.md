## 易错口诀（临考扫一眼）

- **指针**：传值改不了指针本身；`*p++` = `*(p++)`；数组名传址
- **数组**：二维初始化**行可省、列必留**；数组名定义后不能 `=`
- **字符串**：`""` 自动带 `\0`，`{}` 单字符不一定有 `\0`
- **宏**：编译期替换，不占运行时间；`#define d 2` 后不能写 `d++`
- **判断**：`while(1)` 非 0 继续；数组实参传**地址**；C89 无专门逻辑型
